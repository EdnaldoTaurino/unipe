HTML

HTML(linguagem de marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web. A tecnologia é fruto entre os padrões Hytime e SGML. Hytime é um padão para representação estruturada de hipermidia e conteúdo baseado em tempo. Os hipertextos são a junção de vários elementos-Palavras, vídeos e conteudos- que, quando conectados, estabelecem uma rede de dados que permite a comunicaçãp, o armazenamento e o compartilhamento de informações. Por exemplo, ao visitar um website, vocÊ encontra diversas informações com formatações diferentes, como parágrafos, bullets e fontes destintas. Pois então, essa estruturação é realizada por meio do HTML.

Qual a função do HTML?

Imagine que você precisa construir um site do zero com informações da sua empresa nesse caso, você precisará conhecer a estrutura do HTML, criando um documento no bloco de notas e acrescentando codigos- tags -, que indicarão parágrafos, quebra de texto, espaçamento, imagens entre outros. Em geral em um unico site existem diversos documentos HTML, criando páginas como homepage, pagina de produto e informacoes sobre a empresa. Em cada aba existe um código diferente.

Lista de protocolos web

  1. Http
    • É usado toda vez que acessamos um site, um protocolo de comunicação que é usado para transferir dados pela Internet. Ele é utilizado para a comunicação entre clientes (navegadores) e servidores, permitindo que as informações de um website sejam acessadas e exibidas em um navegador da web.
  2. IP
    • IP significa Internet Protocol e é um protocolo de comunicação de rede usado para enviar e receber dados entre dispositivos em uma rede. Cada dispositivo tem um endereço IP único usado para identificá-lo na rede. O IP é um protocolo de camada de rede do modelo OSI e é usado em conjunto com outros protocolos, como o TCP, para fornecer comunicação eficiente entre dispositivos.
  3. FTP
    • Protocolo de rede usado para transferir arquivos de um computador para outro através da Internet. O FTP é uma maneira rápida e fácil de transferir arquivos grandes ou muitos arquivos entre computadores. Funciona através de uma conexão cliente-servidor, o cliente se conecta ao servidor FTP para enviar ou receber arquivos. O FTP é um dos protocolos mais antigos e amplamente utilizados na Internet.
  4. TCP/IP
    • Responsável pelo endereçamento e roteamento dos pacotes de dados. Juntos, o TCP e o IP permitem a comunicação confiável e eficiente entre dispositivos em uma rede, incluindo a Internet. O TCP/IP é amplamente utilizado como base para a comunicação em redes de computadores em todo o mundo.

O que é CSS

CSS é uma linguagem de estilo utilizada para controlar a apresentação visual de um documento HTML ou XML, permitindo que você defina o layout, fontes, cores e outros aspectos de design de suas páginas da web. É composto por regras que definem como os elementos HTML devem ser apresentados na página e essas regras são definidas em um arquivo CSS separado do HTML. Em resumo, o CSS é uma ferramenta poderosa para criar páginas da web estilizadas e profissionais.

O que é HTML

HTML (Hypertext Markup Language) é uma linguagem de marcação utilizada para criar e estruturar conteúdo na web. É a base de toda página da web e permite que os desenvolvedores criem documentos que podem ser interpretados pelos navegadores para exibir texto, imagens, vídeos e outros tipos de conteúdo para os usuários. O HTML usa tags para identificar os elementos da página e definir sua estrutura, como títulos, parágrafos, links, imagens e muito mais. Em resumo, o HTML é a linguagem que permite criar o conteúdo e a estrutura básica de uma página da web.

O que é Java Script

JavaScript é uma linguagem de programação utilizada principalmente para criar interatividade em páginas da web. Ele é interpretado pelos navegadores e permite que os desenvolvedores criem comportamentos dinâmicos na página, como animações, validação de formulários, eventos de mouse e teclado, carregamento assíncrono de conteúdo e muito mais. O JavaScript pode ser utilizado tanto no front-end quanto no back-end, e é uma das linguagens de programação mais populares atualmente. Em resumo, o JavaScript é uma linguagem de programação que permite a criação de interatividade e dinamismo em páginas da web.
